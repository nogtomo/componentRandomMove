# componentRandomMove
選択したコンポーネントにrandom移動を実行します。
▼使用方法
メッシュの任意のコンポーネント（vertex.edge,face）を選択しスクリプトを実行
コントロールウィンドウ上でXYZに数値を設定して実行

## 🎛️ ツールウィンドウの項目説明 (UI Reference)

ツールを起動すると、以下の設定ウィンドウが表示されます。各項目の詳細は以下の通りです。

### 1. Vertex Randomize (頂点のランダム移動設定)
頂点に対してジッター（小刻みなノイズ移動）を加える設定を行います。

| 項目名 (UI Label) | 設定内容・説明 |
| :--- | :--- |
| **Jitter Amount** | 頂点をランダムに移動させる最大距離（移動量）を設定します。スライダーまたは数値入力で直感的に調整可能です。 |
| **Seed (same result)** | ランダムパターンの基準となるシード値です。同じ数値に設定すれば、常にまったく同じランダム形状が再現されます。 |
| **Random** (ボタン) | ワンクリックでシード値をランダムに変更します。異なる形状パターンを素早く試したい時に便利です。 |
| **Lock Axis** | ランダム移動から**除外（固定）する軸**を選択します。<br>・**Lock X / Lock Y / Lock Z** にチェックを入れた軸方向には頂点が移動しなくなります（例：地平を平らに保ちたい場合は `Lock Y` にチェックを入れます）。 |
| **Lock Boundary** | チェックを入れると、メッシュの外周（境界エッジ）にある頂点を移動から除外します。全体のシルエットや外枠の形を崩したくない場合に有効です。 |
| **Use Object Space** | チェックを入れると「オブジェクトスペース」基準、外すと「ワールドスペース」基準で頂点を移動します。回転しているオブジェクトやスロープに沿ってノイズを加えたい場合はオンにします。 |

---

### 2. Triangulate / Random Flip (三角形分割 & エッジフリップ設定)
トポロジー（ポリゴンの接続状態）の変更に関する設定を行います。

| 項目名 (UI Label) | 設定内容・説明 |
| :--- | :--- |
| **Triangulate** | チェックを入れると、頂点移動の後にオブジェクト（または選択フェース）を三角形分割します。<br>※初期値は `OFF`（チェックなし）に設定されています。形状の歪み効果のみを得たい場合はオフのまま実行してください。 |
| **Random Flip Edges** | チェックを入れると、三角形分割されたエッジをランダムな方向にフリップ（対角線の架け替え）し、より不規則で有機的なポリゴン流れを作ります。 |
| **Flip Probability** | エッジフリップが発生する確率（0.0 〜 1.0）を設定します。数値が高いほど、より多くのエッジがランダムに架け替えられます（※ `Random Flip Edges` 有効時のみ）。 |
| **Flip Iterations** | エッジフリップ処理を繰り返す回数を指定します。回数を増やすことで、より複雑に絡み合った不規則な分割パターンが作成されます（※ `Random Flip Edges` 有効時のみ）。 |
| **Keep History** | チェックを入れると、実行後もMayaのヒストリ（インプットノード）を残します。チェックを外すと、実行後に自動でヒストリが削除（クリーンアップ）されます。 |

---

### 3. 操作ボタン
| ボタン名 (Button) | 説明 |
| :--- | :--- |
| **Run on Selected** | 選択しているオブジェクト、または選択しているコンポーネント（頂点/エッジ/フェース）に対して処理を実行します。 |
| **Close** | ウィンドウを閉じます。 |
